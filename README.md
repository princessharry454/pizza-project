# 🍕 Pizza Sales Analysis Dashboard

## 📌 Project Overview
This project is a comprehensive **Pizza Sales Analysis Dashboard** built with **Power BI**, **SQL**, and **Excel** to analyze pizza sales performance and uncover valuable business insights.

The dashboard helps stakeholders:
- Monitor revenue performance
- Understand customer ordering behavior
- Identify top-selling and underperforming pizzas
- Analyze sales by category and size
- Track daily and monthly sales trends

---

## 🎯 Problem Statement
The objective of this project is to analyze pizza sales data and generate insights that support better business decisions in:
- Sales optimization
- Inventory management
- Customer preference analysis
- Product performance evaluation

---

## 🛠️ Tools & Technologies
- Microsoft Excel 2021
- MS SQL Server 19
- SQL Server Management Studio (SSMS)
- Power BI Desktop

---

## 📂 Dataset Overview
The dataset includes:
- Order ID
- Pizza Category
- Pizza Size
- Quantity Ordered
- Unit Price
- Total Price
- Order Date
- Order Time

## 📈 Dashboard Visualizations

### Daily Trend for Total Orders
Analyzes order activity across days to identify demand fluctuations and sales patterns.

### Monthly Trend for Total Orders
Tracks monthly order performance to identify seasonal or recurring trends.

### Sales by Pizza Category
Shows the percentage contribution of each pizza category to total sales.

### Sales by Pizza Size
Highlights customer preferences across different pizza sizes.

### Total Pizzas Sold by Category
Compares pizza categories based on quantity sold.

### Top 5 Best Sellers
Ranks pizzas based on:
- Revenue
- Quantity Sold
- Total Orders

#### Top Revenue Generating Pizzas
- Pizza name column not detected in dataset.


### Bottom 5 Best Sellers
Identifies pizzas with the lowest sales performance.

## 🔍 Business Insights
- High-performing pizza categories contribute significantly to overall revenue.
- Larger pizza sizes tend to generate higher sales.
- Peak order periods provide operational planning opportunities.
- Top-selling pizzas can be prioritized in marketing campaigns.
